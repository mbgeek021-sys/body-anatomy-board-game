# 🫀 Body Anatomy Board Game

Interactive medical anatomy board game with dice rolling, trivia challenges, cards, and multiplayer room system.

---

## 🎮 Game Features

- 🎲 Dice rolling movement
- 🧠 Anatomy-based board (feet → brain)
- 🃏 Card system (good & bad outcomes)
- ❓ Trivia questions (move forward if correct)
- 🛡️ Safe zones & quarantine system
- 👥 Multiplayer room codes

---

## 🧪 Game Goal

Start from the feet and navigate through the human body.  
Reach the **brain** to win the game 🧠🏆
